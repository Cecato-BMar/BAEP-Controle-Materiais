Política de Privacidade

Voltar
Importante: Esta Política de Privacidade descreve como o Sistema de Reserva BAEP coleta, utiliza, armazena e protege as informações pessoais dos usuários.

1. Introdução

O Sistema de Reserva BAEP está comprometido com a proteção da privacidade e dos dados pessoais de seus usuários, em conformidade com a Lei Geral de Proteção de Dados (LGPD) e demais legislações aplicáveis.

Esta Política de Privacidade tem como objetivo informar de maneira clara e transparente como os dados pessoais são tratados no âmbito do sistema.

2. Dados Coletados

O Sistema de Reserva BAEP coleta e processa os seguintes tipos de dados pessoais:

2.1. Dados de Usuários do Sistema
  • Nome completo
  • Nome de usuário (login)
  • Endereço de e-mail institucional
  • Cargo ou função
  • Informações de acesso (data, hora, endereço IP)
  • Registros de atividades realizadas no sistema
2.2. Dados de Policiais Cadastrados
  • Nome completo
  • Registro de Identificação (RE)
  • Informações de contato
  • Histórico de movimentações de materiais
  • Status (ativo, inativo)

3. Finalidade do Tratamento de Dados

Os dados pessoais coletados são utilizados exclusivamente para as seguintes finalidades:

  • Gerenciamento de acesso ao sistema
  • Autenticação e autorização de usuários
  • Registro e controle de movimentações de materiais
  • Geração de relatórios operacionais e gerenciais
  • Auditoria e segurança do sistema
  • Comunicação institucional relacionada ao uso do sistema
  • Cumprimento de obrigações legais e regulatórias

4. Base Legal para o Tratamento de Dados

O tratamento de dados pessoais no Sistema de Reserva BAEP é realizado com base nas seguintes hipóteses legais previstas na LGPD:

  • Execução de contrato ou procedimentos preliminares: para viabilizar o acesso e uso do sistema pelos usuários autorizados;
  • Cumprimento de obrigação legal ou regulatória: para atender às exigências legais relacionadas ao controle de materiais e equipamentos policiais;
  • Exercício regular de direitos em processo: para documentação e comprovação de atividades em caso de processos administrativos ou judiciais;
  • Legítimo interesse: para garantir a segurança, integridade e disponibilidade do sistema, bem como para prevenir fraudes e uso indevido.

5. Compartilhamento de Dados

Os dados pessoais coletados pelo Sistema de Reserva BAEP podem ser compartilhados nas seguintes situações:

  • Com outros órgãos da Polícia Militar, quando necessário para o cumprimento de finalidades institucionais;
  • Com autoridades públicas, mediante requisição legal ou ordem judicial;
  • Com prestadores de serviços técnicos que auxiliam na operação e manutenção do sistema, sempre mediante contratos que garantam a confidencialidade e segurança dos dados.

O Sistema de Reserva BAEP não comercializa, aluga ou transfere dados pessoais para terceiros com finalidades comerciais ou de marketing.

6. Armazenamento e Segurança dos Dados

O Sistema de Reserva BAEP adota medidas técnicas e administrativas de segurança para proteger os dados pessoais contra acessos não autorizados, destruição, perda, alteração ou divulgação indevida, incluindo:

  • Controle de acesso baseado em perfis de usuários;
  • Registro de logs de acesso e atividades;
  • Criptografia de dados sensíveis;
  • Backup periódico das informações;
  • Políticas de senhas fortes e autenticação segura;
  • Treinamento e conscientização dos usuários sobre segurança da informação.

7. Período de Retenção dos Dados

Os dados pessoais são armazenados pelo tempo necessário para cumprir as finalidades para as quais foram coletados, observando-se:

  • Prazos legais de guarda de documentos e informações;
  • Prazos prescricionais aplicáveis a possíveis responsabilidades;
  • Necessidade operacional e histórica dos dados.

Após o término do período de retenção, os dados pessoais serão excluídos ou anonimizados, salvo quando houver obrigação legal de mantê-los.

8. Direitos dos Titulares de Dados

Em conformidade com a LGPD, os titulares de dados pessoais têm os seguintes direitos:

  • Confirmação da existência de tratamento: verificar se seus dados pessoais são tratados pelo sistema;
  • Acesso aos dados: obter informações sobre quais dados pessoais são tratados;
  • Correção de dados incompletos, inexatos ou desatualizados: solicitar a atualização de informações incorretas;
  • Anonimização, bloqueio ou eliminação: quando aplicável e não houver impedimento legal;
  • Informação sobre compartilhamento: saber com quais entidades públicas e privadas os dados são compartilhados;
  • Revogação do consentimento: quando o tratamento for baseado no consentimento e não houver outra base legal aplicável.

Para exercer esses direitos, o titular deve entrar em contato com o Encarregado de Proteção de Dados (DPO) do sistema, conforme informações disponíveis na seção de contato.

9. Cookies e Tecnologias Similares

O Sistema de Reserva BAEP utiliza cookies e tecnologias similares para melhorar a experiência do usuário e o funcionamento do sistema, incluindo:

  • Cookies de sessão: para manter o usuário autenticado durante a navegação;
  • Cookies de preferências: para lembrar configurações e personalização do usuário;
  • Cookies de segurança: para proteger contra acessos não autorizados e atividades fraudulentas.

O usuário pode configurar seu navegador para recusar cookies, mas isso pode afetar o funcionamento adequado do sistema.

10. Alterações na Política de Privacidade

Esta Política de Privacidade pode ser atualizada periodicamente para refletir mudanças nas práticas de tratamento de dados ou alterações na legislação aplicável.

As alterações significativas serão comunicadas aos usuários por meio de avisos no sistema ou por e-mail.

Recomenda-se a consulta regular a esta política para se manter informado sobre as práticas de privacidade do Sistema de Reserva BAEP.

11. Encarregado de Proteção de Dados (DPO)

O Encarregado de Proteção de Dados (Data Protection Officer - DPO) é responsável por garantir a conformidade do Sistema de Reserva BAEP com a legislação de proteção de dados e por atender às solicitações dos titulares de dados.

Para entrar em contato com o DPO, utilize os seguintes canais:

  • Telefone: (11) 3333-5555
  • Endereço: Av. Principal, 1000 - São Paulo/SP - CEP: 01000-000
